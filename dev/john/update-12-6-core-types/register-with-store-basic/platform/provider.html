Basic Store Example

Demonstrate basic store interactions.

Please click the register button to add a store to Storefront (it will start up Storefront and OpenFin Workspace if it isn't already running). Once a store is registered the show/hide buttons will be enabled. Use show to show the store you just created. Deregister will remove your store entry from OpenFin Workspace.

This is a simple example showing how you can hand code a store.
