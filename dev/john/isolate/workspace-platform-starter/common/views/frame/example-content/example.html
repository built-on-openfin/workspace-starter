A Simple Framed Example

This example imports a script framed.js that allows the parent view's title to be updated when this framed content changes it's title (triggered by the button below).

If the fin api is injected into the frame and the preload capability is available then the preload.js script brings in the logic to update the title and the framed.js logic will not run.

This Framed App example is used to demonstrate that you can control whether the fin api is injected into frames and the available options and how you could still have communication between child and parent without the fin api.

The content you try to frame by updating the Framed App Manifest's custom data must support being framed by the origin of the parent.
