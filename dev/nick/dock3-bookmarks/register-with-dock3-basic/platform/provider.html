Basic Dock3 Example

Demonstrate basic dock3 interactions.

Please click the Initialize button to create a Dock3 Provider (it will start up Dock3 and OpenFin Workspace if it isn't already running). Once a dock is initialized it will appear automatically and the minimize and shutdown buttons will be enabled. Use Minmize to minimize the dock you just created. Shutdown will remove your dock entry from OpenFin Workspace.

This is a simple example showing how you can hand code a Dock.
